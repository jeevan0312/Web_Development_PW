HTML5 Features

Structural Elements

HTML5 introduces semantic structural elements, such as:

  • <header>: Defines a header for a document or a section.
  • <nav>: Defines a navigation menu.
  • <section>: Defines a section in a document.
  • <article>: Defines an independent, self-contained content.
  • <footer>: Defines a footer for a document or a section.

Audio and Video Support

HTML5 introduces native support for embedding audio and video content:

  • <audio>: Allows embedding audio content into web pages with controls for playback.
  • <video>: Allows embedding video content into web pages with controls for playback.

Form Enhancements

HTML5 introduces several enhancements to web forms:

  • Input Types: New input types such as email, URL, date, and range provide better input validation and user experience.
  • Attributes: Attributes like placeholder, autofocus, and required enhance form usability and accessibility.
  • Form Elements: New elements like <datalist> and <output> provide more options for form customization and interaction.
